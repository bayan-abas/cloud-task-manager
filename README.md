# TaskFlow

TaskFlow is a cloud-based task management system that helps users create, update, delete, and track tasks.

## What the system does

* User can register and login
* User can add new tasks
* User can edit tasks
* User can delete tasks
* User can view task status

## Technologies

* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot
* Database: MySQL / AWS RDS
* Cloud: AWS
* CI/CD: GitHub Actions
* Monitoring: CloudWatch

## AWS Services

* S3 / CloudFront for frontend hosting
* Elastic Beanstalk or EC2 for backend hosting
* RDS for database
* Route 53 for domain management
* ACM for HTTPS certificates
* CloudWatch for monitoring and logs

## How to Run Locally

1. Clone the repository
2. Open the Frontend folder
3. Run the backend project
4. Connect to the database
5. Open the website in a browser
